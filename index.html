FIFA World Cup 2026

Free, live-updating JSON API for every match

⚡ Auto-updates every 10 minutes

📡 Endpoints

GET matches.json All 104 matches
GET finished.json Completed matches with scores
GET live.json Currently live matches
GET upcoming.json Scheduled fixtures
GET groups.json Matches grouped by stage/group
GET tournament.json Tournament metadata & info

📖 Usage

All endpoints return application/json. CORS is enabled.

fetch('https://h4ck0v3rflow.github.io/fifa-wc-2026-api/matches.json')
  .then(r => r.json())
  .then(data => console.log(data.matches.length, 'matches'));

Data sourced from Bing Sports (SportRadar).
